Data Customizations




Lifestyle and Home Corp is striving to provide their customers a more personalized experience allowing them to add special delivery instructions when placing orders.
  • Customers and Orders must have a custom field to store custom bonus points.
  • Customer gets the option to sign in using credentials.
  • The current cart is merged with the existing cart, if any, found in the customer’s profile.
  • Customers should have the option to opt in for the loyalty scheme.
  • Admins and Marketing team should have the option to update the bonus points scheme.

Steps



Perform the following programmatically, using the SDK:
  1. Create a Custom Type to store bonus points.
  2. Login an existing user and merge the anonymous cart with the existing cart, if any.
  3. Provide an opt in option to customer.
  4. Create a Custom Object to save bonus points calculation scheme.
